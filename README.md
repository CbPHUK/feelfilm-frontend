# FeelFilm

> Эмоциональный дневник о фильмах, сериалах, аниме и книгах

FeelFilm — это платформа, где люди пишут не рецензии и не оценки, а **эмоциональный след**: с чем пришёл к произведению и что унёс после. Два слова до. Два слова после.

---

## Что это такое

Когда смотришь фильм в состоянии тревоги — и выходишь с ощущением покоя — это ценнее любой рецензии. FeelFilm собирает такие моменты и позволяет находить произведения по настроению: не по жанру или рейтингу, а по тому, что ты сейчас чувствуешь и что хочешь почувствовать.

Единая экосистема для фильмов, сериалов, аниме и книг — потому что эмоциональный портрет человека не делится по жанрам.

---

## Возможности

- **Лента** — хронологический поток записей всех пользователей по всем типам контента
- **Поиск по настроению** — выбираешь своё состояние, получаешь список произведений которые другие смотрели именно тогда
- **Каталог** — все произведения с фильтрацией по типу (фильмы / сериалы / аниме / книги)
- **Страница произведения** — агрегированный эмоциональный портрет: с чем приходят и что уносят
- **Профиль** — личный архив записей и эмоциональная статистика
